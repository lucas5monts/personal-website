# Lucas Montoya Personal Website

A simple static personal website for Lucas Montoya. The site includes a home page, project overview, interests page, hobbies page, and contact page.

## Pages

- `index.html` - Home page with a short bio
- `projects2.html` - Personal projects and technical work
- `projects3.html` - Interests in technology, math, AI, finance, and learning
- `hobbies4.html` - Hobbies including sports, fitness, gaming, cars, and cooking
- `contact5.html` - Contact information and email link

## Project Files

- `mystyle.css` - Shared styling for the website
- `script.js` - Light/dark mode toggle
- `*.jpeg`, `*.PNG`, `*.png` - Images used across the site
- `images/` - Additional image folder

## How to View

No installation or build step is needed. Open `index.html` in a web browser to view the site.

You can also run a simple local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Features

- Multi-page HTML website
- Shared navigation across pages
- Custom CSS styling
- Light/dark mode button
- Personal bio, projects, interests, hobbies, and contact information

## Notes

This project is built with plain HTML, CSS, and JavaScript. It does not use any frameworks, package managers, or external dependencies.
